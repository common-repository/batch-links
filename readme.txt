Adds links to posts in a SEO manner, so you can use your WordPress-powered blog to boost the SEO rankings of the linked sites.

== Description ==

This plugin is designed to add links to posts for a predefined list of keywords. 
It is primary utilized as a SEO tool, so you can use your WordPress-powered blog 
to boost the SEO rankings of the other websites, which you link through the 
keywords.

== Installation ==

1. Download the installation archive and uncompress
1. Upload the unarchived folder to your plugins directory (this is wp-content/plugins/)
1. Activate the plugin in your WordPress administration panel (this is the Plugins page)
1. Configure the plugin from the Options page in the WordPress administration panel
1. That’s it ;)
